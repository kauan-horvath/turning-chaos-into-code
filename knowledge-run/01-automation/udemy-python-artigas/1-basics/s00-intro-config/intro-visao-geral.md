**Primeiro Teste (Sem IDE):**
```python
for i in range(10):
    print("Hello World")

# Necessário mudar a extensão do arquivo de .txt para .py
```

- **Como checar se o Python está instalado:**
  - No terminal/CMD: `python --version`

- **Comandos básicos no CMD:**
  - `cd nome_da_pasta`: Entrar em diretórios.
  - `cls`: Limpar a tela do terminal.
  - `python arquivo.py`: Executar o script (Teste realizado com `usando-txt.py` — Sucesso).
  - `exit()`: Sair do interpretador Python.

- **Configuração do VS Code:**
  - Lembrar de baixar as extensões necessárias (Python da Microsoft).
  - Adicionar pastas e diretórios ao Workspace para facilitar o trabalho.
